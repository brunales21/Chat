---------------------------------------
Registering -> REGISTER your-nickname
Channel creation commands -> CREATE #channel
Add contact command -> CREATE user
Join a channel -> JOIN #channel
Quitting from a channel -> PART #channel
Deleting a contact -> DELETE nickname
Message to channel command -> PRIVMSG #channel:message
Message to user command -> PRIVMSG user:message
Command to list contacts -> LU
Command to list channels -> LC
